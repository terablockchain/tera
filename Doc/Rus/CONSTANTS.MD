## Примечание:
* Константы хранятся в формате JSON
* В логических константах, таких как включение или отключение чего либо, рекомендуется использовать число 0 или 1
* Любую константу можно изменить из командной строки следующим образом (запускать внутри каталога ~/wallet/Source):
    ```
    node set ИМЯ_Константы=Значение
    ```
* Некоторые константы (такие как номер порта ноды) используется только при старте ноды, поэтому для их применения требуется перезапустить ноду
